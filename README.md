## react-redux-sample

①create-react-appインストール
~~~
npm install -g create-react-app
~~~

②雛形の作成
~~~
create-react-app redux-calculator
~~~

③アプリの起動確認
~~~
cd redux-calculator
npm start
~~~
これで「http://localhost:3000」にアプリが起動している。

④react-redux,reduxインストール
~~~
npm install react-redux --save
npm install redux --save
~~~
